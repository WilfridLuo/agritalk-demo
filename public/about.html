AgriTalk-RL Demo Notes

Project

AgriTalk-RL frames agricultural consultation as a sequential decision process: first decide whether evidence is enough, then either clarify or respond.

Demo design

The main demo page is intentionally stripped down for immersion. This page keeps the background explanation separate from the live product experience.

Images

Real crop photos are linked from Wikimedia Commons. Cases are preset to simulate uploaded evidence in a polished product flow.

Deploy

This project is static HTML, CSS, and JavaScript. It can be deployed directly to Vercel or shared temporarily via a tunnel.
